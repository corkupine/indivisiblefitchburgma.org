# Indivisible Fitchburg MA Website

This is a static website for the Indivisible Fitchburg chapter.

## Project Structure
- `index.html` - Main HTML file
- `styles.css` - CSS styles
- `scripts.js` - JavaScript functionality
- `images/` - Directory for images (needs to be created)

## Running Locally

### Setup
First, install dependencies:

```bash
npm install
```

### Development Server
Start the development server with:

```bash
# No caching (recommended for development)
npm run dev

# OR standard server
npm start
```

Then visit `http://localhost:8080` in your web browser.

### Method 3: Using Visual Studio Code
If you use VS Code, you can install the "Live Server" extension and click "Go Live" in the bottom right corner.

## Image Placeholders
The site currently uses placeholder references for images. Create an `images` directory and add:
- `hero-background.jpg` - For the hero section background
- `group-action.jpg` - For the about section

You can use images from the national Indivisible site or your own appropriate images.

## Deployment
To deploy this site to a web server, simply upload all files to your hosting provider.

## Customizing Content
Update the HTML file to change text content, events, and other information specific to your chapter.
